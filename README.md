### 2025

- **[AgentCon - HR Copilot](2025/AgentCon%20-%20HR%20Copilot/)**
  
  Building enterprise HR copilots using Microsoft Copilot Studio. Presented at AgentCon 2025 Perth, featuring two specialized agents (Priya for policy guidance and Logan for leave management) with RAG-based search and Azure OpenAI integration.

- **[FabCon Vienna Updates](2025/FabCon%20Vienna%20Updates/)**
  
  Microsoft Fabric FabCon Vienna 2026 announcements and updates shared with Perth Microsoft Data & Analytics User Group. Covers platform updates, security enhancements, developer experiences, and new data integration capabilities.

- **[Global AI Bootcamp](2025/Global%20AI%20Bootcamp/)**
  
  Getting started with Phi-4 and Chainlit for building AI applications. Hands-on workshop delivered at Global AI Bootcamp Perth, featuring local model deployment, multimodal capabilities, and practical demo implementations.

- **[Intro to AI Functions In Microsoft Fabric](2025/Intro%20to%20AI%20Functions%20In%20Microsoft%20Fabric/)**
  
  Introduction to Microsoft Fabric's built-in AI Functions for data enrichment and transformation. Covers 7 core AI functions (fix_grammar, analyze_sentiment, summarize, translate, similarity, classify, mask) with practical datasets and hands-on examples.

- **[PMI - AI for Project Managers](2025/PMI%20-%20AI%20for%20Project%20Managers/)**
  
  Boosting productivity with Microsoft AI for project management. Presented at PMI Western Australia Chapter, featuring Microsoft Copilot across Teams, M365, Planner, and Azure AI for project planning, execution, and governance. Includes practical prompt templates for cost-benefit analysis, EVM calculations, risk management, and agile user stories.

- **[UWA - AI Insights and Application in the Industry](2025/UWA%20-%20AI%20Insights%20and%20Application%20in%20the%20Industry/)**
  
  AI insights and practical applications in enterprise. Presented at UWA Data Science Club, covering RAG (Retrieval-Augmented Generation) architecture, multi-agent AI systems, Azure AI platform capabilities, and real-world case studies like Fujitsu's 67% productivity improvement in sales automation.

- **[Industry Inner Join v15](2025/Industry%20Inner%20Join%20v15/)**
  
  Future-proofing data estates for AI transformation with Microsoft Fabric. Presented at Industry Inner Join Perth, covering the evolution from 20+ separate Microsoft data products to the unified Fabric platform. Features OneLake zero-ETL architecture with 19 data sources, Fabric Data Agents, Gartner Magic Quadrant leadership, and special certification opportunities (100% discount for DP-700 & DP-600).

### 2023
