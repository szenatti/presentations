- **No data movement** - Data stays in source systems
- **Reduced complexity** - Eliminate ETL pipelines
- **Real-time access** - Query data where it lives
- **Cost savings** - No duplicate storage costs

## Fabric Data Agents

AI-powered data agents that act as virtual analysts for your enterprise data.

### What are Fabric Data Agents?

A virtual analyst that allows users to interact with enterprise data in OneLake using natural language.

### Key Capabilities

1. **Seamless Reasoning** ⭐ New
   - Reason over data in OneLake
   - Include mirrored databases
   - Create powerful data experts tailored to your data domain

2. **Agent Operations (AgentOps)** ⭐ New
   - Support for data agents in Git
   - Deployment pipelines integration
   - Streamlined AI development lifecycle

3. **External Invocation** ⭐ New
   - Invoke data agents externally via APIs
   - Detailed responses including SQL query execution steps
   - Integration with enterprise applications

### Data Sources for Agents

Data agents can interact with:
- **Lakehouse** - Apache Spark analytics
- **Warehouse** - SQL-based data warehouse
- **Eventhouse** - Real-time analytics data
- **Semantic Models** - Power BI datasets
- **Mirrored Databases** - Zero-copy databases

### Integration Points

#### Consumer Applications
- **Microsoft 365** - Integrate with M365 apps
- **Teams** - Data insights in Microsoft Teams
- **Copilot Studio** - Build custom copilot experiences
- **Azure AI Foundry** - Advanced AI scenarios
- **Custom Endpoints** - API-based integrations
